# 🌙 Low-Light Image Enhancement

## 📌 Overview

This project enhances low-light images using the DUAL Illumination Enhancement algorithm. It improves image brightness and visibility while preserving important details.

---

## 🚀 Features

- Enhance low-light images
- Supports JPG, PNG and BMP images
- Batch image processing
- Fast enhancement using Python
- Automatic output generation

---

## 🛠 Technologies Used

- Python
- OpenCV
- NumPy
- SciPy
- tqdm

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/SuryaRamisetty/Low-light-Image-Enhancement.git
```

Go to the project folder

```bash
cd Low-light-Image-Enhancement
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶ Run the Project

```bash
python demo.py
```

---

## 📸 Output

The enhanced images are automatically saved inside

```
demo/enhanced/
```
